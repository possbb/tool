# 时间列自动转行工具

网页版表格转换工具，可自动识别表格中代表时间的列，并把这些列转成“时间 / 值”长表。

## 在线使用

打开 GitHub Pages 地址后，直接上传表格即可。文件在浏览器本地处理，不会上传到服务器。

## 本地网页版

双击 `启动网页版.bat` 后访问：

```text
http://127.0.0.1:5000
```

## 支持格式

- 输入：`.xlsx`、`.xls`、`.xlsm`、`.csv`、`.tsv`
- 输出：`.xlsx`

## 可自动识别的时间列名示例

- `2024-01`
- `2024/01/31`
- `2024年1月`
- `2025年`
- `2025`
- `Q1 2024`
- `2024 Q1`
- `Jan 2024`

## 命令行使用方法

```powershell
pip install -r requirements.txt
python .\time_columns_to_rows.py "你的表格.xlsx"
```
