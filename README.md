# 时间列自动转行工具

## 推荐使用方式

### 方式 1：在线版

https://possbb.github.io/tool/

### 方式 2：离线网页版（发给别人最方便）

1. 解压压缩包
2. 双击 `打开离线网页版.bat`
3. 选择表格并转换

说明：离线网页版不需要安装 Python，文件在浏览器本地处理。

## 如果离线网页版遇到复杂 Excel 报错

请使用本地服务版：

1. 双击 `首次使用先安装依赖.bat`
2. 双击 `启动网页版.bat`
3. 在打开的网页里上传表格

## 支持格式

- 输入：xlsx、xls、xlsm、csv、tsv
- 输出：xlsx

## 可识别的时间列示例

- 2024-01
- 2024年1月
- 2025年
- 2025
- Q1 2024
- Jan 2024
