# 打包一键 EXE

生成 Windows 一键版：

```powershell
$py='C:\Users\weiyuchen\.cache\codex-runtimes\codex-primary-runtime\dependencies\python\python.exe'
& $py -m pip install pyinstaller
& $py -m PyInstaller --noconfirm --clean --onefile --name "时间列自动转行工具" --add-data "templates;templates" --collect-submodules pandas --collect-submodules openpyxl --collect-submodules flask --collect-submodules werkzeug desktop_app.py
```

生成结果：

```text
dist\时间列自动转行工具.exe
```
